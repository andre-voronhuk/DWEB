    -- Podemos definir Ancoras em nossa pagina para navegar dentro de nosso arquivo
    Assim como esta sendo utilizado em "texto" e "imagem",
    a sintaxe funciona da seguinte forma:

    escolhemos palavras para serem o titulo de nosso hyperlink
    colocando elas dentro da tag "a", porem precisamos definir onde estará a 'ancora',
    então vamo até o elemento desejado e colocamos uma ancora que nada mais
    é que um "a"(mas nao necessariamente a tag "a", pode ser uma imagem ou qualquer elemento)
    com o parametro "name" setado, como "texto" por exemplo, após fazer isso, voltamos a palavra que
    sera o titulo do hyperlink e no parametro "href" colocamos "#NomeDoParamName"
    nesse caso: href="#texto"


    MAIS UM SEGREDO! 🤫🤫:
    podemos em nosso arquivo CSS ou na tag "HTML", setarmos o parametro style
    para "scroll-behavior: smooth;" dessa forma, a navegador da nossa pagina sera mais agradavel
    e quando usarmos uma ancora dentro da pagina, haverá um scroll suave até o destino
    e nao uma troca repentina que é o comportamento padrão,
    essa pagina esta usando o que foi citado acima, portando voce pode testar clicando em algum hyperlink

    1999 - Versão 4.01
    Suporte para opções multimídias, folha de estilos e melhorar as práticas de marcação
    2014 - Versão HTML 5
    Aperfeiçoar a experiência da World Wide Web para os desenvolvedores e usuários finais

    TAGS

    As tags definem ao navegador a marcação de conteúdo.
    As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

    As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

    Arquivos para Download

    -- Aqui temos um exemplo de como fazer o download de arquivos
    funciona da seguinte forma: colocamos o caminho do arquivo em "href"
    nesse caso href="img.zip", usamos target="_blank" para abrir em outra guia
    e o tipo em type="application/zip"
    dentro da tag "a" colocamos a palavra para identificar para o usuário
    que é ali onde ele deve clicar para fazer o download dos arquivos
